Information

Pour poursuivre le développement du jeu ou pour le modder, vous pouvez cloner le repository git. Cliquez ici. Pour ce faire, nous vous conseillons d’utiliser le logiciel Git Kraken qui vous permettra de gérer vos branches en même temps. Le logiciel contient une version gratuite pour utilisation non-commerciale, ce qui profitera certainement à plusieurs modders. Vous aurez aussi besoin de la dernière version de l’éditeur Unity pour continuer à travailler sur ce projet (que vous pouvez obtenir ici Unity).
